# CalculadoraIMC

A Calculadora IMC é um aplicativo Android desenvolvido em Kotlin, criado para calcular o Índice de Massa Corporal de forma simples, rápida e precisa.
O usuário informa o peso e a altura, e o app exibe tanto o valor do IMC quanto a classificação correspondente baseada nos padrões da Organização Mundial da Saúde (OMS).


### Funcionalidades

* Inserção de peso e altura
* Validação inteligente dos campos
* Cálculo do IMC
* Classificação automática do resultado
* Exibição detalhada em uma segunda tela
* Exibi classificação (baixo peso, normal, sobrepeso, obesidade)
* Interface simples e intuitiva
* Código claro, organizado e fácil de manter
* Navegação entre telas utilizando Jetpack Navigation Compose
* Gerenciamento de estado com StateFlow

### Tecnologias utilizadas

* Kotlin
* Android Studio
* Jetpack Compose
* ViewModel
* StateFlow
* Navigation Compose
